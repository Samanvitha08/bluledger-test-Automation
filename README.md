BluLedger Test Automation Framework

 Overview
This project is a Test Automation Framework for BluLedger application using:
- Java
- Selenium WebDriver
- TestNG
- Maven



 Framework Design
- Page Object Model (POM)
- Data-driven testing using Excel
- Retry mechanism for failed tests
- Extent Reports for reporting


 Project Structure
- base → Base classes
- pages → Page Object classes
- tests → Test cases
- utils → Utilities (Excel, Config, Screenshot)
- listeners → TestNG listeners


 Features
- Cross-browser support
- Screenshot capture on failure
- Retry failed test cases
- Detailed HTML reports
